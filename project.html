Parent Linker

TOOL : Java

Parent Linker, a cutting-edge Java-based application, revolutionizes student record management by streamlining attendance tracking, grade management, and leave administration. This intuitive platform securely stores student data and automatically sends timely SMS updates to parents on attendance, grades, and important school events. By enhancing parental engagement and improving student accountability, Parent Linker fosters a collaborative educational environment.

Anti-theft flooring system with NodeMCU

TOOL : IOT

The IoT-based anti-theft flooring system, powered by NodeMCU, is a revolutionary security solution that detects movement, alerts users, and captures real-time images of intrusions. Integrated pressure sensors under the flooring trigger instant alerts to users' mobile devices via Wi-Fi connectivity, providing visual verification through image capture. This innovative system offers customizable alert settings, remote monitoring, and enhanced security for homes, offices, and sensitive areas, effectively deterring potential intruders and providing peace of mind.
